# RAG Constructor

**Low‑Code платформа для создания RAG‑ассистентов на основе внутренних документов компании**

Проект позволяет бизнес‑аналитикам, менеджерам и другим сотрудникам без навыков программирования самостоятельно настраивать пайплайны обработки документов (PDF, HTML, CSV) и получать готовый чат‑интерфейс для вопросов по загруженным материалам. Вся конфигурация выполняется через веб‑интерфейс без написания кода.

## Какую проблему решает проект

- **Снижение порога входа в RAG‑технологии** – для создания чат‑бота по документам больше не нужно писать Python‑скрипты.
- **Ускорение получения ответов** – от загрузки файла до первого вопроса проходит менее часа.
- **Контроль над данными** – возможность работы с локальными LLM без отправки конфиденциальных документов во внешние облака.

## Ключевые возможности MVP

- Загрузка документов через веб‑интерфейс (PDF, TXT)
- Настройка параметров чанкинга (`chunk_size`, `overlap`)
- Выбор модели эмбеддингов и LLM из предустановленного списка
- Автоматическая индексация в векторной БД Qdrant
- Чат‑интерфейс для общения с загруженными документами
- Полное разделение Control Plane (настройки) и Data Plane (рантайм инференс)
