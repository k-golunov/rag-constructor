Советы по настройке

  • Размер чанка влияет на точность поиска. Меньший чанк — точнее, но больше запросов к API.
  • Перекрытие помогает не потерять контекст на границах чанков. 10–20% от размера — хорошее начало.
  • text-embedding-3-small — быстрый и дешёвый вариант для большинства задач.
  • gpt-4o-mini — оптимальный баланс качества и стоимости для RAG.
