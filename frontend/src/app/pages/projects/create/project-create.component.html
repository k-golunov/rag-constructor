Советы

  • Размер чанка — меньший чанк точнее, но дороже. Начните с 800.
  • Перекрытие — 10–20% от размера чанка предотвращает потерю контекста.
  • Абзацы — лучший вариант для большинства документов.
  • URL API оставьте пустым для облачных провайдеров. Для Ollama укажите http://localhost:11434/v1.
  • API ключ можно передать глобально через переменную окружения.
